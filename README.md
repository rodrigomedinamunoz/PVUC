 # Prototipo de Plataforma Veterinaria Unificada y Centralizada

  Este es el código correspondiente a un prototipo de una Web de atención veterinaria unificada y centralizada.

  ## Levantando la aplicación
  En un terminal ejecutar los siguientes comandos.
  
  `npm i` para instalar las dependencias.

  `npm run dev` para abrir el servidor de desarrollo.
